# Wix-Demo

A WiX Toolset v5 `.msi` installer example for a .NET 8.0 application.

## Caution

This project only works on **Windows**. WiX Toolset does not support macOS or Linux.

## Tools Required

- **Visual Studio 2022** (VS2022)
- **HeatWave for VS2022 Extension**
- **.NET 8.0 SDK**
